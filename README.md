# Five Nights at Freddy's 2: The Patrox Way
Another version of FNaF 2, including "All Max" Button for custom night, loading screens instead of a black screen and some other utils

# Installation
The Patrox Way is now available in HTML5 format, but if you want to download it to play in .exe format click on this to download the standalone

(HTML5 Format does **NOT** include perspective objects so if you want the best experience, download de .exe format)
